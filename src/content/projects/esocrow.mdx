## Intro

ESO Crow was created to help navigate the [Elder Scrolls Online](https://www.elderscrollsonline.com/) (without using way shrines). This can be quite handy when youâ€™re new to the game or starting a new character.

## Features

- Navigate Elder Scrolls Online using without using Way shrine Navigation. (Uses [Dijkstra's algorithm](https://en.wikipedia.org/wiki/Dijkstra%27s_algorithm) for Navigation using [NetworkX](https://networkx.org/))
- List of [Locations](https://esocrow.com/locations) and [Zones](https://esocrow.com/zones). With all possible connections from each [Location](https://esocrow.com/Windhelm)
- Contact form using WTForms w/ Recaptcha

## Built Using

**Front:** HTML, CSS and a little JavaScript.

**Back:** Python with [Flask](https://flask.palletsprojects.com/) and [NetworkX](https://networkx.org/)

**Python Libraries used:**

- Flask
  - Flask-Mail
- NetworkX
- WTForms
