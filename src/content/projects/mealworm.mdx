## Intro

Mealworm helps you organise your meals and meal plans, it generates a shopping list from your plans to help speed up making a shopping list.

## Features

- Meal Planning
- Shopping List Generator
- Login, Registration, Authentication System
- DRF API

## Front-end built with

[React JS](https://reactjs.org/), JavaScript, HTML and CSS.

**Libraries Used**:

- [Chakra UI](https://chakra-ui.com/)
- [Axios](https://axios-http.com/) (for communicating with DRF backend using HTTP requests).
- [JOI](https://joi.dev/) (for Form Validation)

## Back-end built with

Python, [Django](https://www.djangoproject.com/), [Django Rest Framework](https://www.django-rest-framework.org) and MySQL.

**Frameworks:**

- [Django](https://www.djangoproject.com/)
  - [Django Rest Framework](https://www.django-rest-framework.org)
  - [DRF Nested Routers](https://github.com/alanjds/drf-nested-routers)
  - [Djoser](https://djoser.readthedocs.io) for JWT Authentication

**Testing:**

- [Model Bakery](https://model-bakery.readthedocs.io)
- [Pytest](https://pytest.org)
