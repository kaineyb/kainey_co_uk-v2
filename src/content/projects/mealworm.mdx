Mealworm helps you organise your meals and meal plans, it generates a shopping list from your plans to help speed up making a shopping list :)

Deploys from Github using a cpanel yaml and custom deploy.sh

Powered, protected and certified by Cloudflare :)

---

**Front end:** React, JS, HTML and CSS.

**Backend:** Python, Django, Django Rest Framework and MySQL.

**Features:**

- Meal Planning
- Shopping List Generator
- Login, Registration, Authentication System
- DRF API

**Libraries used:**

- Backend
  - Django & DRF
  - DRF Nested Routers
  - Djoser for JWT Authentiction
  - Model Bakery and Pytest for Testing
- Frontend
  - Chakra UI
  - Axios for communicating with DRF backend.
  - JOI for Form Validation
